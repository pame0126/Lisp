***Shell Lisp***
Construir un intérprete de expresiones del lenguaje de 
programación funcional Lisp utilizando C.

**Valores**
- Numeros Naturales.
- Boleanos --> #t #f

**Funciones Aritmetica**
- +, -, *, modulo, quotient.

**Funciones Relacion**
- =, <, >, <=, >=.

**Funciones Logicas**
- and, or, not.

*Nota:* Las funciones deben trabajar de forma anidada,
        sobre literales o expresiones.
        Salida --> Ctrl+D
